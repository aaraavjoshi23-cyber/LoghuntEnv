# LogHuntEnv — AI Cybersecurity Environment

## What it does
An AI agent that detects network attacks in real time,
like a smart security guard for computers.

## Attacks detected
- DDoS
- PortScan  
- Bot
- Benign traffic

## How to run
1. Double click RUN_ME.bat
2. Wait 5-10 minutes
3. See results!

## Actions the AI can take
- PASS - do nothing
- FILTER_IP - block suspicious IP
- CORRELATE - investigate pattern
- PIVOT_USER - flag user account
- ALERT - raise security alert
- QUARANTINE - full lockdown

## Reward
- Catching real attack = +10
- False alarm = -5
- Kill chain detected = +20
